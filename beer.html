About the Design

使用活潑粗體字來營造歡樂的氛圍，網站主色採用logo的藍色，同時搭配補色的黃色以增加色彩對比。

活発で太字のフォントを使用して楽しい雰囲気を演出し、ウェブサイトのメインカラーはロゴの青色を採用し、同時に補色の黄色を添えて色彩対比を増やしています。

Using lively bold fonts to create a cheerful atmosphere, the main color of the website adopts the blue from the logo, while incorporating complementary yellow to enhance color contrast.
